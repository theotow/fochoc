## Installation

1. ``` make build ```
2. ```mv ./fochoc_(win|mac|linux) /usr/local/bin/fochoc```
3. use ```$ fochoc```

## Supported Exchanges
- binance
- kraken
- poloniex
- bittrex

## Supported Coins / Tokens
